Lambda expressions

Java 

OptionalInt sum =
  IntStream.of(1, 2, 3, 4, 5)
    // 1, 3, 5
  .filter(i -> (i % 2) != 0)
  // 2, 4, 6
  .map((i) -> { i++; return i; })
  // 2 + 4 + 6
  .reduce((i, a) -> a + i);
ES6 (name here is Arrows) 

let sum =
  Array.of(1, 2, 3, 4, 5)
  // 1, 3, 5
  .filter(i => (i % 2) === 0)
  // 2, 4, 6
  .map((i) => { i++; return i; })
  // 2 + 4 + 6
  .reduce((i, a) => i + a);

Yeeeeah! Java don't give up
